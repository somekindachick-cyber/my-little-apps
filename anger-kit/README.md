# Anger Emergency Kit

8-bit 像素风情绪释放工具  
按住按钮释放怒气，守护者降临并安慰你。

## 功能
- 按住红色按钮蓄力
- 像素爆炸动画
- 守护者出现 + 安慰语
- 每日 / 每月生气记录
- Firebase 跨设备同步
